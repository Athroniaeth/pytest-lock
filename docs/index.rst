Welcome to pytest-lock's documentation!
=======================================

Introduction
------------
Pytest-lock est un plugin pytest qui permet une gestion avancee des tests avec des fixtures flexibles et des arguments CLI personnalisables. Il est conçu pour simplifier et etendre les capacites de test en offrant des fonctionnalites X et Y.

Prerequis
---------
Ce plugin necessite Python 3.6+ et pytest 5.0+. Assurez-vous que ces prerequis sont satisfaits avant l'installation.

Installation
------------
Pour utiliser pytest-lock, installez-le avec pip :

.. code-block:: console

   pip install pytest-lock
